# 콘텐츠 전략

서기(현)를 "글 몇 개 있는 아카이브"가 아니라 철도기술사 준비생과 현업 종사자가 검색해서
찾아오는 아카이브로 키우기 위한 초기 콘텐츠 계획입니다. 글쓰기 규칙 자체는
[content/README.md](../content/README.md)를 참고하세요. 이 문서는 "무엇을, 어떤 순서로,
얼마나 자주 쓸 것인가"를 다룹니다.

## 1. 카테고리별 콘텐츠 비율

`news`는 시의성 있는 뉴스 브리핑이라 특성이 다르므로 별도로 관리하고,
나머지 5개 카테고리는 아래 비율을 목표로 합니다.

| 카테고리   | 비율  | 성격                                   | 역할                                  |
| ---------- | ----- | --------------------------------------- | -------------------------------------- |
| `news`     | 필요 시 (별도) | 철도 정책·개통·산업 동향               | 재방문 유도, 홈 히어로 공급            |
| `study`    | 40%   | 12개 분야 시험 대비 학습 자료            | 검색 유입의 주력, 사이트의 핵심 정체성 |
| `terms`    | 25%   | 용어 사전형 (질문형 키워드 저격)         | 롱테일 검색 유입, 최소 노력 대비 최고 효율 |
| `standard` | 15%   | 설계기준 핵심 수치·규정 정리             | 실무자·전공자 검색 유입                |
| `case`     | 10%   | 실제 시공 사례                          | 체류시간, 실무 신뢰도 확보             |
| `resource` | 10%   | 학습법, 시험 전략 등 참고 자료           | 신규 방문자 전환, 브랜드 신뢰도        |

`terms`는 분량이 짧고 검색 의도가 명확("OOO란", "OOO 뜻")해 가장 적은 노력으로
검색 유입을 만들 수 있는 카테고리입니다. `study`는 사이트의 정체성 그 자체이므로
가장 많은 비중을 두고 꾸준히 채워야 합니다.

## 3. SEO 키워드 구조

- **타겟팅 원칙**: 한 글은 하나의 "메인 키워드"만 명확히 겨냥합니다. 여러 키워드를 욱여넣지 않습니다.
- **키워드 배치**: `title`(글 제목), `description`(frontmatter), 본문 첫 문단, 최소 1개의 `##` 소제목에
  메인 키워드를 자연스럽게 포함시킵니다.
- **검색 의도별 카테고리 매핑**:
  - "OOO란/뜻" (정의형) → `terms`
  - "OOO 기준/수치" (규정형) → `standard`
  - "OOO 사례/시공" (실무형) → `case`
  - "OOO 종류/원리/장단점" (학습형) → `study`
  - "OOO 하는 법/전략" (하우투형) → `resource`
- **롱테일 우선**: 경쟁이 큰 "철도설계" 같은 단일 키워드보다 "철도기술사 궤도틀림 종류"처럼
  구체적인 롱테일 키워드를 우선 타겟팅합니다. 초기 도메인 권위가 낮을 때 더 빨리 순위가 잡힙니다.
- `tags` frontmatter는 곧 부제 키워드 역할을 합니다. 관련 검색어를 태그로 등록해두면
  관련 글 추천([internal-linking.md](./internal-linking.md) 참고)에도 그대로 활용됩니다.

## 4. 발행 운영 구조

지속 가능한 최소 단위로 주간 캘린더를 고정합니다.

| 요일 | 카테고리         | 비고                                   |
| ---- | ----------------- | ----------------------------------------- |
| 월   | study              | 새로운 분야 학습 자료                       |
| 화   | terms              | 짧고 빠르게 용어 1~2개                      |
| 수   | standard 또는 case | 격주 교차 발행                              |
| 목   | study              | 학습 자료 보강 (기존 분야 심화)             |
| 금   | resource 또는 news | 학습법/전략 또는 그 주 철도 뉴스 정리        |

- 위 20개 리스트 소진 후에는 각 글의 "함께 보면 좋은 개념" 링크에서 파생되는 하위 주제를
  다음 글감으로 잡습니다 (예: "궤간이란"을 쓰고 나면 다음 글감은 "슬랙이란").
- 최소 기준선은 **주 3회(study 1회 + terms 1회 + standard/case/resource 중 1회)**로 잡아도
  검색엔진 크롤링 빈도 유지에는 충분합니다. 완전히 멈추는 것만 피하세요.
