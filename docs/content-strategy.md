# 콘텐츠 전략

분명한경제를 "글 몇 개 있는 사이트"가 아니라 검색 유입이 쌓이는 미디어로 키우기 위한
초기 콘텐츠 계획입니다. 글쓰기 규칙 자체는 [content/README.md](../content/README.md)를 참고하세요.
이 문서는 "무엇을, 어떤 순서로, 얼마나 자주 쓸 것인가"를 다룹니다.

## 1. 카테고리별 콘텐츠 비율

`today`는 매일 발행되는 뉴스 브리핑이라 특성이 다르므로 별도로 관리하고,
나머지 4개 카테고리(`learn`/`invest`/`terms`/`blog`)는 아래 비율을 목표로 합니다.

| 카테고리 | 비율  | 성격                                   | 트래픽 역할                          |
| -------- | ----- | -------------------------------------- | ------------------------------------- |
| `today`  | 매일 1개 (별도) | 시황 브리핑, 최신성이 핵심             | 재방문 유도, 홈 히어로 공급           |
| `learn`  | 35%   | 경제 초보 대상 기초 개념                | 검색 유입의 주력 (검색량 많은 입문 키워드) |
| `invest` | 30%   | 투자 입문 + 분석 프레임워크             | 체류시간·전환(뉴스레터/광고) 기여      |
| `terms`  | 25%   | 용어 사전형 (질문형 키워드 저격)         | 롱테일 검색 유입, 최소 노력 대비 최고 효율 |
| `blog`   | 10%   | 의견/전망/브랜드 스토리                 | 브랜드 신뢰도, 시리즈 콘텐츠           |

`terms`는 분량이 짧고 검색 의도가 명확("OOO란", "OOO 뜻")해 가장 적은 노력으로
검색 유입을 만들 수 있는 카테고리입니다. 초기에는 `terms` 비중을 다소 높게 잡아도 좋습니다.

> 매일 반복 적용하세요.

## 3. SEO 키워드 구조

- **타겟팅 원칙**: 한 글은 하나의 "메인 키워드"만 명확히 겨냥합니다. 여러 키워드를 욱여넣지 않습니다.
- **키워드 배치**: `title`(글 제목), `description`(frontmatter), 본문 첫 문단, 최소 1개의 `##` 소제목에
  메인 키워드를 자연스럽게 포함시킵니다.
- **검색 의도별 카테고리 매핑**:
  - "OOO란/뜻" (정의형) → `terms`
  - "OOO 하는 법/방법" (하우투형) → `learn`
  - "OOO 추천/비교/분석" (의사결정형) → `invest`
  - "오늘/최근 OOO" (시사형) → `today`
- **롱테일 우선**: 경쟁이 큰 "주식투자" 같은 단일 키워드보다 "주식투자 처음 시작하는 법"처럼
  구체적인 롱테일 키워드를 우선 타겟팅합니다. 초기 도메인 권위가 낮을 때 더 빨리 순위가 잡힙니다.
- `tags` frontmatter는 곧 부제 키워드 역할을 합니다. 관련 검색어를 태그로 등록해두면
  관련 글 추천([internal-linking.md](./internal-linking.md) 참고)에도 그대로 활용됩니다.

## 4. "매일 글 1개" 운영 구조

지속 가능한 최소 단위로 주간 캘린더를 고정합니다. 매일 1개, 주 7개 발행을 기본값으로 합니다.

| 요일 | 카테고리        | 비고                                   |
| ---- | --------------- | ---------------------------------------- |
| 월   | today           | 주말 시황 정리 + 이번 주 주목 이슈 예고     |
| 화   | today           | 당일 이슈 브리핑                          |
| 수   | learn 또는 terms | 격주 교차 발행                            |
| 목   | today           | 당일 이슈 브리핑                          |
| 금   | invest          | 주간 투자 인사이트                         |
| 토   | today           | 주간 이슈 총정리(약식)                     |
| 일   | blog 또는 terms | 격주 교차 발행 (일요일은 가벼운 주제 권장)  |

- 위 20개 리스트 소진 후에는 "왜 중요한가" 섹션에서 파생된 하위 주제를 다음 글감으로 잡습니다
  (예: "PER란"을 쓰고 나면 다음 글감은 "PER와 PBR 같이 보는 법").
- 매일 발행이 부담되면 최소 기준선은 **주 5회(today 3회 + learn/invest/terms 중 2회)**로 낮춰도
  검색엔진 관점에서는 크롤링 빈도 유지에 충분합니다. 완전히 멈추는 것만 피하세요.
